📖 基本ルール概要

Objective ゲーム目標

8ポイント（チーム戦では11ポイント）を先に獲得したプレイヤーの勝利。ポイントは戦場の制圧（Conquer）と保持（Hold）で獲得する。

Deck Building デッキ構築

メインデッキ40枚 + ルーンデッキ12枚 + 戦場カード3枚。レジェンドカードが決めるドメインに従ってデッキを構築する。

Turn Structure ターン構造

①アウェイクン（レディ化）→ ②ホールド（ポイント獲得）→ ③チャネル（ルーン配置）→ ④ドロー → ⑤アクションフェーズ → ⑥ターン終了

Showdown ショーダウン

ユニットを敵が占拠している戦場に移動させると発生する戦闘。双方のユニットのMightの合計でダメージを与え合い、生き残った側が戦場を制圧する。

Resource System リソースシステム

ルーンはエグゾースト（タップ）するとエネルギー1を供給。リサイクル（デッキに戻す）するとそのドメインのパワーを供給する。

Final Point Rule 最終ポイントルール

最後の8点目は、その同じターンに他のすべての戦場からもポイントを獲得している場合のみ、単一の戦場の制圧で獲得できる。
